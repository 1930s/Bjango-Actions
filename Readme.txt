----------------------- 1.3 -

Some awesome Photoshop Actions and Automator Workflows for iOS app and icon designers. For a full list of what the Photoshop Actions and Workflows do, please visit http://bjango.com/articles/actions/

** Bjango Actions 1.3.atn **
The full set of Photoshop Actions. The default set.

** Bjango Actions 1.3 with pattern.atn **
The full set of Photoshop Actions. Scaling Actions also reset pattern scales, using the Scale Patterns to 100% script (http://bjango.com/articles/scalepatternsto100/). You'll need to install the Scale Patterns to 100% script for these to work.

** Bjango Actions (Old).atn **
Old and outdated Photoshop Actions. Might be handy if you're using Photoshop CS5 or earlier.
